# Wifi Deauth Detector

WiFi deauthentication attack detector.

Pendeteksi serangan deautentikasi WiFi.


## Description

Monitors wireless traffic to detect deauthentication frame floods used in DoS attacks.

## Deskripsi

Memantau lalu lintas nirkabel untuk mendeteksi banjir frame deautentikasi yang digunakan dalam serangan DoS.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Monitor: sudo python deauth_detector.py --interface wlan0
- Demo: python deauth_detector.py --demo

## Penggunaan

- Monitor: sudo python deauth_detector.py --interface wlan0
- Demo: python deauth_detector.py --demo
